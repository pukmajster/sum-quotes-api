
## Description

A NestJS backend for my Quotes project. The API handles all CRUD aspects for the frontend (e.g. login, registration, posting quotes, updating profile, ...) while also generating new "quotes of the day".

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start:dev

# production mode
$ npm run start:prod
```
